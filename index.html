    代码规范与风格

    这是我自己总结的一套代码风格，主要是对typescript与react使用，具有一个规范的代码思路，提升开发效率

    Vite 构建脚手架

    Vite是Vue团队提供的一款构建工具，是基于Rollup进行封装整合，更加快速且简单的打包构建工具

    Webpack 构建脚手架

    第一代构建工具，其生态非常庞大，对待问题有各式各样的解决方案

    Redux 原理与使用

    Redux是一款状态管理工具，其出名的是优秀的设计理念，多样的生态，不依赖其他框架，完全解耦，是现在最为流行的状态管理工具之一

    Typescript 的使用

    Typescript是由微软推出的一款基于js的强类型语言，其主要用于js库的编写，具有非常高的稳定性，对项目代码可以起到稳定作用

    数据结构与算法

    数据结构与算法是程序员进阶的必修课，计算机技术更迭就是算法与数据结构的突破，只为追求更快更强

    Next.js 构建脚手架

    其优秀的SSR方案，让前端开发者可以更加专注于业务开发，更好的支持SEO，主要使用在H5和官网

    Taro 构建小程序

    Taro 是一个开放式跨端跨框架解决方案，支持使用 React/Vue/Nerv 等框架来开发 微信 / 京东 / 百度 / 支付宝 / 字节跳动 / QQ / 飞书 小程序 / H5 / RN 等应用。

    Promise 原理与使用

    Promise是一种异步编程的解决方案，其主要解决了回调地狱的问题，其本质是一个状态机，对于异步编程有着非常重要的作用
