    使用指南 💡文档

    Vite 构建脚手架

    Vite是Vue团队提供的一款构建工具，是基于Rollup进行封装整合，更加快速且简单的打包构建工具

    Webpack 构建脚手架

    第一代构建工具，其生态非常庞大，对待问题有各式各样的解决方案

    Redux 原理与使用

    Redux是一款状态管理工具，其出名的是优秀的设计理念，多样的生态，不依赖其他框架，完全解耦，是现在最为流行的状态管理工具之一

    Typescript 的使用

    Typescript是由微软推出的一款基于js的强类型语言，其主要用于js库的编写，具有非常高的稳定性，对项目代码可以起到稳定作用

    React 原理与构建

    解析React的构建原理，探索react核心数据流

    数据结构与算法

    数据结构与算法是程序员进阶的必修课，计算机技术更迭就是算法与数据结构的突破，只为追求更快更强

    Markdown 增强

    新增文字对齐、上下角标、脚注、标记、任务列表、数学公式、流程图、图表与幻灯片支持

    幻灯片页面

    添加幻灯片页面以显示你喜欢的内容

    布局增强

    添加路径导航、页脚、改进的导航栏、改进的页面导航等。

    浏览量与评论

    配合 Waline 来开启阅读量统计与评论支持

    文章信息

    为你的文章添加作者、写作日期、预计阅读时间、字数统计等信息

    博客支持

    使用一些很棒的布局提供博客功能，通过文章的日期、标签和分类，自动生成文章、分类、标签与时间轴列表

    主题色切换
